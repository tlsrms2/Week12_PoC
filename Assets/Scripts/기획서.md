# 게임 기획서: 빛의 3원색 리듬 퍼즐 (가제)

## 1. 게임 개요 및 핵심 컨셉
* **장르:** 리듬 + 색채 퍼즐
* **핵심 컨셉:** 빛의 3원색(RGB) 가산 혼합 메커니즘을 음악적 템포와 결합한 비동기식 매칭 게임
* **게임 방식:** '콜 앤 리스폰스(Call & Response)'를 재해석하여, 음악 비트에 맞춰 제시되는 시각적 패턴을 플레이어가 실시간 또는 제한 시간 내에 동일하게 구현하는 방식

---

## 2. 화면 구성 및 아트 스타일
* **레이아웃:** 화면 중심을 기준으로 좌우 대칭형 그리드 구조
    * **좌측 (선생님 영역):** 시스템이 음악에 맞춰 정답 패턴을 칠하며 보여주는 공간
    * **우측 (플레이어 영역):** 플레이어가 실제로 마우스와 키보드를 조작하여 색을 채우는 입력 공간

---

## 3. 조작 및 색상 혼합 시스템

### 3.1 기본 조작
* **색상 입력:** 키보드 키(`Q`, `W`, `E`)를 누른 상태에서 마우스를 이동(드래그)하면 마우스 커서 경로에 있는 삼각형 타일에 해당 색상이 입혀짐.
    * **`Q` 키:** 빨강 (Red)
    * **`W` 키:** 파랑 (Blue)
    * **`E` 키:** 초록 (Green)
* **수정 및 초기화:** 마우스 **우클릭** 시 해당 칸의 색상이 완전히 지워지며 초기 상태로 리셋됨.

### 3.2 7색 혼합(가산 혼합) 메커니즘
빛의 3원색 조합을 바탕으로 총 7가지의 색상 표현이 가능하며, **'동시 입력'**과 **'덧칠'** 방식을 모두 지원함.

| 조합 키 / 방식 | 결과 색상 | 시각적 특징 |
| :--- | :--- | :--- |
| **Q** | 🔴 빨강 (Red) | 기본 원색 |
| **W** | 🔵 파랑 (Blue) | 기본 원색 |
| **E** | 🟢 초록 (Green) | 기본 원색 |
| **Q + W** (또는 빨강 위에 파랑 덧칠) | 🟣 마젠타 (Magenta) | 혼합색 |
| **W + E** (또는 파랑 위에 초록 덧칠) | 🔷 시안 (Cyan) | 혼합색 |
| **Q + E** (또는 빨강 위에 초록 덧칠) | 🟡 옐로우 (Yellow) | 혼합색 |
| **Q + W + E** (또는 모든 색 덧칠) | ⚪ 화이트 (White) | 최종 혼합색 (가장 강하게 발광) |

---

## 4. 핵심 플레이 루프 및 판정 규칙

```
[1. 패턴 제시 (선생님 턴 시작)] ──> 플레이어 실시간 입력 가능
           │
           ▼
[2. 선생님 턴 종료 및 유예 시간] ──> 제한 시간 동안 최종 수정 및 완성
           │
           ▼
[3. 단일 시점 최종 판정] ─────────> 타이머 종료 순간 좌우 그리드 일치 여부 체크
           │
           ▼
[4. 그리드 리셋] ─────────────────> 다음 파트(턴)로 전환
```

1. **패턴 제시 (선생님 턴):** 음악이 재생되면 비트에 맞춰 좌측(선생님) 타일이 순차적으로 음악과 싱크되어 칠해짐.
2. **플레이어 자유 입력:** 플레이어는 선생님의 턴이 시작됨과 동시에 우측 그리드에 색을 칠할 수 있음. (선생님의 연출을 실시간으로 따라 하거나, 패턴을 다 본 뒤 조작하는 등 자유로운 전략 가능)
3. **유예 시간 및 마감:** 선생님의 패턴 제시가 끝난 후 수 초간의 유예 시간이 주어지며, 음악 비트에 맞춘 타이머가 마감 시점을 시각적으로 직관적으로 안내함.
4. **단일 시점 판정:** 제한 시간이 정확히 끝나는 그 순간(단일 시점)에 좌측과 우측의 그리드 상태를 비교하여 최종 성공/실패 및 등급 판정을 내림.
5. **라운드 리셋:** 판정이 종료되면 양쪽 그리드의 색상이 모두 지워지고 다음 파트(턴)로 전환됨.
